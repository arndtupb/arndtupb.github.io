# arndtupb.github.io

This site is now authored with Quarto.

## Edit homepage content

1. Update `/home/runner/work/arndtupb.github.io/arndtupb.github.io/index.qmd`.
2. Render HTML with:

   ```bash
   quarto render
   ```

3. Commit both `index.qmd` and the generated `index.html`.

This avoids manually editing `index.html`.
